# 🍅 React Pomodoro & Meditation Timer

Простой, быстрый и удобный таймер для работы и медитации. Создан для тех, кто ценит фокус и минимализм.

**Попробовать онлайн: [открыть сайт](https://echidnartem.github.io/pomodoro-timer/)**

## 📋 О проекте
Многие онлайн-таймеры перегружены рекламой, требуют регистрации или не сохраняют прогресс при закрытии вкладки. Этот проект — личный инструмент, который всегда под рукой, работает быстро и сохраняет все данные локально в вашем браузере.

## ✨ Ключевые возможности
* **Два режима:** Классический Pomodoro (25/5) и режим медитации (10).
* **Менеджер задач:** Встроенный список задач с возможностью добавления, редактирования и удаления.
* **Сохранение данных:** Состояние таймера, текущие задачи и выбранная тема сохраняются в `localStorage`.
* **Умные уведомления:**
    * Звуковой сигнал по окончании цикла.
    * Отображение времени и статуса (Работа/Отдых) прямо в заголовке вкладки.
* **Адаптивность:** Удобный интерфейс как для десктопа, так и для мобильных устройств.
* **Точность:** Использование `Date.now()` гарантирует, что таймер не замедляется, когда вкладка находится в фоновом режиме.

## 🛠 Технологический стек
* **React 19** (**Vite**) — современная база приложения.
* **Vite Plugin SVGR** — позволяет импортировать SVG-файлы как готовые React-компоненты (используется для главной иллюстрации).
* **Lucide React** — библиотека стильных и легких иконок.
* **Custom Hooks** — модульная архитектура:
    * useTimer: ядро логики отсчета времени.
    * useTimerPersistence: автоматическая синхронизация с localStorage.
    * useTimerModes: управление пресетами времени и состоянием сессий.
* **LocalStorage API** — для сохранения списка задач, текущей темы и прогресса таймера.

## 🚀 Как запустить локально

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/echidnartem/pomodoro-timer.git
    ```
2.  Перейдите в папку проекта:
    ```bash
    cd pomodoro-timer
    ```
3.  Установите зависимости:
    ```bash
    npm install
    ```
4.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```
