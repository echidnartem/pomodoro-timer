# 🍅 Pomodoro Timer
Простой и удобный таймер Pomodoro, созданный для личного использования. Никакой рекламы, никаких ограничений — только работа и отдых.

# 📋 О проекте
Я пользовался разными онлайн-таймерами Pomodoro, но многие из них либо перегружены рекламой, либо требуют постоянного доступа в интернет, либо не сохраняют историю. Этот проект родился из желания иметь простой, быстрый и надёжный инструмент для личного использования, который всегда под рукой.

# ✨ Функциональность
✅ Классические циклы Pomodoro: 25 минут работы / 5 минут отдыха

✅ Управление таймером: старт, пауза, сброс

✅ Звуковое уведомление по окончании цикла

✅ Счётчик завершённых помидорок с сохранением в LocalStorage

✅ Переключение темы (светлая/тёмная)

✅ Адаптивный дизайн — удобно как на компьютере, так и на телефоне

# 🚀 Установка и запуск
Можно запустить локально:

```bash
# Клонировать репозиторий
git clone https://github.com/echidnartem/pomodoro-timer.git

# Перейти в папку проекта
cd pomodoro-timer

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173
